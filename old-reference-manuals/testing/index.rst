======================
PloneTestCase tests
======================

.. admonition:: Description

    This tutorial will explain how to write safer, better code that makes you
    look more professional. That's right - it's time to write tests, for
    everything you do. Don't worry, it's not boring or complicated, you just
    need to learn how.

.. warning ::

    Technologies presented here are for Plone 3.x and older.

For `up-to-date testing best pratices read about plone.app.testing <https://pypi.python.org/pypi/plone.app.testing>`_.
